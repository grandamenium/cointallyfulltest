# Crypto Tax Estimator

## Overview

**Crypto Tax Estimator** is a web application built with **Next.js**, **React**, and **Tailwind CSS** that helps users estimate their annual crypto tax liability.  
It allows users to input their filing status, income, and crypto gains to calculate estimated federal and state taxes based on the latest tax brackets.
